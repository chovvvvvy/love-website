# 💕 情侣网站 - 使用说明

## 🚀 快速开始

### 打开网站

**方式一：双击打开**
```
双击 index.html 文件
```

**方式二：浏览器打开**
```
右键 index.html → 打开方式 → 选择浏览器
```

## ✨ 功能介绍

### 📱 主要功能

1. **恋爱计时器**
   - 显示相爱的天数、小时、分钟、秒
   - 实时更新

2. **144件小事**
   - 恋爱必做的144件小事清单
   - 打卡记录完成情况
   - 分类显示（旅行、日常、趣味）

3. **纪念日提醒**
   - 添加重要纪念日
   - 自动提醒即将到来的纪念日
   - 显示倒计时

4. **照片墙**
   - 上传照片记录美好瞬间
   - 支持图片和视频
   - 查看大图、编辑、删除

5. **回忆录**
   - 写下心情和故事
   - 记录重要时刻

## 💾 数据存储

### 当前配置：本地存储模式

- ✅ 照片存储在浏览器本地
- ✅ 无需服务器
- ✅ 隐私安全
- ✅ 离线可用

⚠️ **重要提示**：
- 照片只保存在当前浏览器中
- 清除浏览器数据会丢失所有照片
- 换浏览器或设备照片不共享
- **建议定期导出备份**

## 🔧 数据备份

### 导出数据

1. 打开网站
2. 找到"导出数据"按钮
3. 选择导出格式：
   - **ZIP格式**：包含照片和所有数据
   - **JSON格式**：仅包含数据和清单
4. 保存到本地

### 导入数据

1. 打开网站
2. 找到"导入数据"按钮
3. 选择之前导出的备份文件
4. 恢复数据

## 🆘 常见问题

### Q: 照片丢失了怎么办？

**A**: 如果有导出的备份文件，可以导入恢复。如果没有备份，很遗憾无法恢复。建议定期备份。

### Q: 换电脑了怎么办？

**A**: 将整个项目文件夹复制到新电脑，然后导入备份数据即可。

### Q: 清除缓存后数据还在吗？

**A**: 不会。清除浏览器数据会删除所有本地存储的照片和清单。请谨慎操作。

### Q: 可以分享给对方看吗？

**A**: 当前模式下，照片只存在你的浏览器中，对方看不到。如需共享，需要导出数据发送给对方，或启用服务器模式。
