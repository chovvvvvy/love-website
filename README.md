# 💕 情侣网站

一个纪念美好时光的专属网站

## ✨ 功能

- ✅ **恋爱计时器** - 记录相爱天数
- ✅ **144件小事** - 共同完成的美好回忆
- ✅ **纪念日提醒** - 重要日期不会忘记
- ✅ **照片墙** - 记录美好瞬间
- ✅ **回忆录** - 写下心情和故事

## 🚀 使用方法

### 直接打开

双击 `index.html` 文件，在浏览器中打开即可使用。

### 本地模式（当前配置）

照片存储在浏览器本地：
- ✅ 无需服务器
- ✅ 隐私安全
- ✅ 离线可用

⚠️ **注意**：照片只保存在本地浏览器中，清除浏览器数据会丢失，建议定期导出备份。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主页面 |
| `config.js` | 配置文件（当前为本地模式） |

## 💾 数据备份

### 导出数据

1. 打开网站
2. 点击"导出数据"按钮
3. 选择导出格式（ZIP或JSON）
4. 保存到本地

### 导入数据

1. 打开网站
2. 点击"导入数据"按钮
3. 选择备份文件
4. 恢复数据

## 🔧 配置

### 切换到服务器模式

如果需要多人共享照片，可以：

1. 修改 `config.js` 中的 `SERVER_MODE = true`
2. 运行服务器（需要配置Python环境）
3. 其他人通过局域网访问

但当前配置为本地模式，更适合个人使用。

## 📝 说明

- 当前模式：**本地存储模式**
- 照片位置：浏览器本地
- 访问方式：双击打开HTML文件
- 数据隐私：完全私密

## 🎉 使用建议

1. 定期导出数据备份
2. 不要清除浏览器缓存
